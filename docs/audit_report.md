## Текущее состояние функционала
- **GUI**: Каркас создан, но логика кнопок (например, "Обновить") не реализована. Запуск в песочнице ограничен (нет дисплея).
- **CLI**: Файл `cli.py` существует, но содержит только заглушку (нужно реализовать парсер).
- **Word COM**: Базовая обертка есть, но требует доработки по жизненному циклу (использование `DispatchEx`).
- **PDF**: Базовый экспорт и слияние реализованы, требуют ужесточения обработки ошибок и структурирования результатов.

## Выявленные проблемы
1. `cli.py` не выполняет полезных действий.
2. Кнопка "Обновить" в GUI не связана с поиском файлов.
3. Отсутствует разделение GUI на виджеты и контроллеры (все в `main_window.py`).
4. Нет CI (GitHub Actions).
5. Тесты не имеют маркеров для разделения интеграционных (Windows/Word) и модульных тестов.
