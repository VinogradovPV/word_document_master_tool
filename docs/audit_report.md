# Отчет об аудите проекта (стабилизация MVP)

## Общая информация
- **Дата**: 2026-06-25
- **Версия Python**: 3.12.3
- **Окружение**: Linux Sandbox (без поддержки Word COM)

## Результаты проверок
- **compileall**: Успешно.
- **pytest**: Успешно (все тесты проходят в Linux).
- **ruff**: Успешно (линтер настроен и пройден).

## Актуальное состояние функционала
- **GUI**:
    - Метод `_refresh_list` частично реализован: сканирует папку и заполняет Treeview.
    - Требуется полная связка с сервисами обработки.
    - Кнопки "Обработать файлы" и "Слияние документов" пока не имеют назначенных команд (`command`).
- **CLI**:
    - Точка входа `wdmt` объявлена в `pyproject.toml`.
    - `cli.py` содержит базовый парсер, но требует доработки для поддержки реальных операций.
- **Word COM**:
    - Обертка `WordApp` переведена на `DispatchEx` и безопасный жизненный цикл.
    - Функционал **не проверен** в среде Windows (требуется ручное тестирование).
- **PDF**:
    - `PdfMergeService` переведен на использование структурированной модели `PdfMergeResult`.

## Выявленные проблемы и ограничения
1. GUI еще не имеет полной связки с сервисами.
2. Кнопки действий в GUI не активны.
3. CLI выполняет только информационные функции (версия, помощь).
4. Отсутствие реальных тестов Word COM в Linux-песочнице.
