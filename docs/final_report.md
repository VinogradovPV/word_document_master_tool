# Итоговый отчет по проекту: Word Document Master Tool (Python)

## Обзор
Проект представляет собой Python-аналог VBA-инструмента для пакетной обработки документов Word, их слияния, нумерации и экспорта в PDF. Инструмент ориентирован на работу в среде Windows с установленным Microsoft Word.

## Реализованный функционал
1. **GUI (Tkinter/ttk)**:
   - Модульная архитектура с использованием кастомных виджетов.
   - Управление списком документов (добавление, удаление, изменение порядка).
   - Интерактивный выбор папок через диалоговые окна.
   - Отображение прогресса выполнения в реальном времени.
2. **CLI**:
   - Точка входа `wdmt` для базовых операций и проверки версии.
3. **Word COM Automation**:
   - Защищенный жизненный цикл приложения Word (изолированные экземпляры через `DispatchEx`).
   - Оптимизированное открытие документов с подавлением диалоговых окон.
   - Подготовка копий: принятие правок, удаление комментариев.
4. **Слияние и Разделение**:
   - Слияние нескольких документов в один с выбором типа разрыва.
   - Разделение объединенных документов на основе уникальных маркеров `WMT_START/WMT_END`.
5. **Нумерация**:
   - Сквозная нумерация страниц в колонтитулах с настройкой шрифта и выравнивания.
   - Управление нумерацией сносок (сквозная или по разделам).
6. **PDF Сервисы**:
   - Экспорт исходников в режиме Read-Only.
   - Слияние сгенерированных PDF-файлов в один через `pypdf`.
7. **Логирование и Отчетность**:
   - Технический лог `application.log` для отладки.
   - Пользовательский отчет `processing_log.tsv` в формате, дружелюбном к Excel (UTF-8 with BOM).
   - Интеграция сканера надстрочных знаков для выявления страниц, требующих проверки.

## Технический стек
- **Язык**: Python 3.12
- **Библиотеки**: `pywin32` (COM), `pypdf` (PDF Merge), `tkinter` (GUI).
- **Качество**: `ruff` (линтер), `pytest` (тесты с моками для Linux).

## Инструкция по запуску
1. Установите зависимости: `pip install .`
2. Запустите GUI: `wdmt-gui` или `python -m word_document_master_tool.gui.app`
3. Для работы в консоли: `wdmt --help`
